网络最大流与最小截问题 -- EK算法详解

图论 EK算法 最大流 最小截
ACM Notes
Publish Date:   2020-07-02
Update Date:   2020-08-15
Word Count:   1.6k

解决问题

对于一张图, 我们对于每条边有两个特性, 单位时间能传输的最大流量(容量)和流量, 流量必须小于等于容量并且对于每个点来说, 流入的流量和与流出的流量和必须相等. 最大流算法就是解决图中一个点到另一个点的单位时间内能传输的最大流量.

重要概念与定理

  • 可行流

    可行流是针对整个图来说的, 对于整个无向图来说, 所有弧满足流量必须小于等于容量并且使得其两端点合法(出入相等), 则我们称从起点流向终点的流量可行.

  • 弧(边)
    1. 饱和弧(容量 = 流量)
    2. 非饱和弧(流量 < 容量)
    3. 零流弧(无流量)
    4. 非零流弧(流量 > 0)
  • 增广链(基于图当前的一种可行流来说的)

    定义: 对于一种可行流方案来说, 起点到终点的一条链(链中的边是没有方向的见图解举例的链, 链是针对整个图来说的, 而路径是针对一个特定传输情况来说的), 其中的所有正向弧都非饱和, 所有反向弧都非零, 则我们称这条链为增广链.

    定理1: 对于一种可行流方案来说, 如果当前方案中还存在增广链, 那么其总流量将还能得到增广(正向弧加流量, 反向弧减流量, 所以要求正向弧非饱和, 反向弧非0), 使得其单位时间的流量更大.
    定理2: 动态相互影响, 当当前可行流方案中的一条增广链被增广为非增广链时, 那么图中调整前的其他增广链, 现在就可能不是增广链了.
    定理3: 当当前可行流方案中不再存在增广链, 那么该图起点的流出流量就必定为最大流.

  • 截集与截量

    定义:

    1. 截集是指我们将图中的节点分为两个集合(V1, V2, V1 ∩ V2为空集, V1 ∪ V2为V), 并且S位于V1, E位于V2, 所有由V1指向V2的有向边的集合, 我们称之为该划分下的图的截集.
    2. 截量就是指截集中的所有边的容量之和.
      性质: 在图中删去截集之后, 起点到终点将不存在路(不连通 -> 割边(桥)).
      定理1(最大流最小截(割)定理): 网络的最大流量等于网络的最小截量.
      定理2: 网络中的任一可行流的流量小于等于在其方案下的任一截集的截量. (那么该图的最大流量就等于最小的截量, 自然也就小于了其他截集的截量了)

算法分析

  • 标号法(EK算法)求解最小截集与最大流
    1. 算法思想: 通过找当前可行流中的增广链并对其增广, 直到可行流中无增广链为止, 那么当前可行流为最大流, 且以已标记点为V1, 未标记点为V2, 由V1连向V2的所有边的集合即最小截集.
    2. 算法实现: 对于一般的最大流问题, 我们开始的每条边的流量是没有给出的, 只给出了容量, 然而标记算法求最大流, 是通过增广链增广的, 所以我们开始的图的流量必须保证是一个可行流, 那么我们怎么构造一个尽量接近最大流的初始可行流呢? 对于每条边赋予尽量大的合法流量或者把所有的边流量全初始化为0, 即零流(但是不推荐, 因为这样很远离最大可行流的流量, 当前可行流流量为0). 构造完之后, 对于每个走过的点我们记录下两个特性: 1. 最大可调整流量, 2. 其上一个点的编号, 首先我们将起点标号为(∞, Vs), 我们对标号点集的所有弧(除开父边)进行延伸, 检查是否存在一条未标记点流向它的非零反向弧或流出至未标记点的非饱和弧(增广链中对弧的要求). 有的话, 我们选择可以更快走到终点的点, 然后标记该点, 继续寻找点集的弧, 当点集的所有弧都不符合的时候, 此时的可行流为最大流, 否则直到终点被标记. 此时被标记的一条链则为一条增广链. 我们对其进行增广, 那么如何增广呢? 我们对每条弧的流量正加反减流量x, x为增广链中的所有可调整流量y(正向弧(当前节点(已标号点集)指向未标号节点): 容量 - 当前流量, 反向弧(未标号节点指向当前节点): 当前流量)的绝对值的最小值.

现实应用

  1. 计算机交互
  2. 电路输送 -> 最大流
  3. 攻防问题 -> 最小截

    我们将点两两之间的多条路转化为一条路, 路的数量转化为路的权重, 截断路使得两个点不能连通, 即最小截集. 图的最大流就等于最小截集的截量, 权值和即路数量的和, 所以问题转化为了求图的最大流.
    陆地岛屿单向边: A, F换一下位置就行了, 根据题意只需求一边即可.
    而对于具体的截集边, 我们在遍历最大流方案下的所有已标记点集中的点, 分别遍历其出边, 如果e[i]未被标记则将i输出即可.
  4. 旅行问题 -> 最小截

    车辆必过的边的集合其实就是截集, 这里的截量就是费用, 所以这道题就是求最小截, 即最大流.

问题拓展

如何通过扩容扩大最大流?
在最大流问题中, 我们最大流 = 最小截, 所以扩大最大流就是扩大最小截集的截量, 即截集中的边的容量.
